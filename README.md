# Bin2Dec Converter 🔢

A simple Binary to Decimal converter built with Java Swing.

## About
This is my first Java GUI project as a beginner developer.
I built this while learning Java in Eclipse IDE.

Project idea from: [florinpop17/app-ideas](https://github.com/florinpop17/app-ideas)

## What I learned
- Java Swing (JFrame, JPanel, JLabel, JTextField, JButton)
- Event handling with ActionListener
- Binary to decimal conversion using Math.pow()
- Layout managers (BoxLayout, BorderLayout, GridBagLayout)
- Styling components with colors and fonts

## How to run
1. Open Eclipse IDE
2. Create a new Java project
3. Copy Bin2DecUI.java into the src folder
4. Run the file!

## Features
- Enter up to 8 binary digits
- Converts binary to decimal instantly
- Shows error message for invalid input
